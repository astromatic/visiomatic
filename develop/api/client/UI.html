    new UI(baseLayers, optionsopt) → {UI}

    Base class for VisiOmatic dialog controls.

    Parameters

    • baseLayers Array.<VTileLayer>

      Array of layers

    • options object <optional>

      Options.

      Properties

      • title string <optional>
         'a control related to VisiOmatic'

        Layer title. Defaults to the basename of the tile URL with extension removed.

      • collapsed boolean <optional>
         true

        Start dialog in the collapsed state?

      • position boolean <optional>
         'topleft'

        Position of the dialog on the map.

    See

    • Leaflet API reference for additional control options.

    Returns

    • UI

      VisiOmatic UI instance.
